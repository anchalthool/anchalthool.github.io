Overview

This project is a fully functioning MERN websitethat lets creators build surveys and polls, share them, and analyze responses. It includes secure authentication, poll voting for logged‑in users, and results visualization with clean bar and chart views.

Highlights

Secure login & protected routes

Create & manage surveys and polls

Users can vote on available polls

Live results via charts & bars

Responsive, snappy React UI

Architecture

Simple three‑tier design with token‑based auth and REST APIs.

React SPA - collects inputs and calls REST APIs.

Express API - exposes endpoints for auth, surveys, polls, and voting.

Auth - issues/verifies tokens and guards protected routes.

MongoDB - stores users, survey/poll definitions, and votes.

Key End‑User Flows

Sign up / Log in via secure portal.

Create a new Survey or Poll with questions/options.

Share or publish the poll; authenticated users can vote.

View results as live charts (bars, percentages, totals).

Run Locally

Assumes frontend/ (React) and backend/ (Node/Express) folders.

# Backend
cd backend
npm install
cp .env.example .env  # set MONGODB_URI, JWT_SECRET
npm run dev

# Frontend
cd ../frontend
npm install
npm start
