IoT Health Monitoring with Voice-Recognized Prescriptions

View on GitHub

Overview

A voice-enabled healthcare web app that lets doctors dictate prescriptions, converts speech to text, extracts key medical details using lightweight NLP, and stores results for later viewing. The platform includes basic authentication and separate experiences for doctors and patients.

Tech Stack

Layer Tech Used
FrontendReact
BackendNode.js, Express
AuthenticationJWT / Sessions
DatabaseMongoDB
VisualizationChart.js / Recharts
ServerExpress (development server)

Key Features

  • Voice-to-Text: Doctors dictate or upload audio and convert it to text using Speech-to-Text APIs.
  • NLP Extraction: Parses the transcript to identify medicine names, dosage, frequency, and instructions.
  • Role-Based Access: Separate registration, login, and dashboards for doctors and patients.
  • Maintainable Structure: Organized routes and processing modules to keep the code clean and extendable.

How It Works

  1. Doctor logs in and records / uploads a voice prescription.
  2. The app converts speech into text using Speech-to-Text.
  3. NLP logic extracts structured prescription fields.
  4. The prescription is displayed instantly and optionally stored in the database.
  5. Doctors and patients can view results in their dashboards.
