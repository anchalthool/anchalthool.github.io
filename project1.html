E-Health NLP (Voice-Based Prescription App)

Overview

E-Health NLP is a voice-enabled web application that allows doctors to dictate medical prescriptions, which are then transcribed, processed using Natural Language Processing (NLP), and optionally stored or displayed. The platform also supports basic user management and separate dashboards for patients and doctors.

Tech Stack

LayerTech Used
FrontendReact
BackendNode.js, Express
AuthenticationJWT / Sessions
DatabaseMongoDB (Compass)
VisualizationCharting library (Chart.js / Recharts)
ServerNode / Express development server

Key Features

Voice-to-Text: Doctors can upload recorded voice (or dictate) prescriptions, which are transcribed using Google's Speech Recognition API.

NLP Parsing: Transcribed text is processed to extract structured information like medicine name, dosage, frequency, etc.

User Roles: Doctors and Patients have separate registration and dashboards.

Modular Code: Separated routes, templates, and processing logic for clean maintainability.

How It Works (Simple Flow)

Doctor logs in and records/upload voice prescription.

App transcribes audio → text using Google STT API.

Text is parsed using simple NLP logic (listen()).

Extracted prescription is shown on screen or saved to DB.

Patient/Doctor can view dashboards.
